# 开发指导

## Skill开发规范

### 目录结构

```
skill-xxx/
├── README.md       # 使用说明（必须）
├── templates/      # 模板文件
├── scripts/        # 辅助脚本
└── examples/       # 示例输出
```

### README格式

```markdown
# Skill名称

## 功能简介
简要描述Skill用途

## 使用命令
ai-skill run xxx --project <项目名>

## 参数说明
| 参数 | 说明 | 必填 |
|------|------|------|
| --project | 项目名称 | 是 |

## 输出示例
展示典型输出格式

## 扩展点
标记可扩展部分
```

---

## 扩展指南

### 可扩展部分
- 模板内容可自定义
- 输出格式可调整
- 参数可新增

### 不可动部分
- 核心处理逻辑
- 必填参数定义
