# 搜索语法说明

## 基本搜索

在公共能力页面的搜索框中输入关键词，系统会匹配以下字段：
- Skill名称
- 功能描述
- 开发阶段
- 标签

## 搜索规则

| 规则 | 说明 | 示例 |
|------|------|------|
| 单关键词 | 匹配任意字段包含该词 | `需求` |
| 多关键词 | AND逻辑，所有关键词都必须匹配 | `AI 生成` |
| 不区分大小写 | 英文关键词大小写不敏感 | `API` = `api` |
| 部分匹配 | 支持部分字符串匹配 | `测试` 匹配 `单元测试` |

## 示例

| 搜索词 | 结果 |
|--------|------|
| `需求` | 返回所有需求阶段或描述包含"需求"的Skill |
| `AI 辅助` | 返回标签包含"AI辅助"的Skill |
| `测试 生成` | 返回测试阶段且是生成类的Skill |
| `代码` | 返回名称或描述包含"代码"的Skill |

## 标签过滤

点击标签按钮可按标签过滤Skill列表：
- **AI辅助**：使用AI技术辅助开发
- **自动化**：自动执行重复任务
- **模板类**：提供标准化模板
- **生成类**：自动生成文档或代码

点击标签后，列表只显示包含该标签的Skill。再次点击可取消过滤。
